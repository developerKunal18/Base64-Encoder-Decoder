# Base64 Encoder / Decoder

A Python script to encode and decode Base64 text.

## Description
This project converts normal text into Base64 format and decodes Base64 back into readable text.

## Features
- Encode text
- Decode Base64
- Useful in APIs and web systems

## Technologies Used
- Python
- base64 module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python base64_tool.py

4. Choose encode or decode

## Example

Input:
hello

Output:
Encoded: aGVsbG8=
